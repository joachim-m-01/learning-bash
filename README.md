# Learning Bash

Ce dépot regroupe des scripts bash que j'ai fait pour apprendre le langage.

## Utilisation

- allow_disallow_kdeconnect.sh: Permet sur opensuse d'activer ou de désactiver kdeconnect. (-h)
- check_passwd.sh:  Vérifie le mot de passe d'un utilisateur en fonction d'un hash
- counter.sh: Minuteur en bash ( exercice formation LFCS Sander Van Vught)
- get_process_number.sh: Le script prend un argument;  le nom de l'utilisateur. Permet de connaitre le nombre de processus actif pour un utilisateur
- update-suse.sh: Permet de mettre à jour mon système
